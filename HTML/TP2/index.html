Manipulation des balises HTML et des sélecteurs CSS

Voici venu le moment d"experimenter les usages des différentes balises HTML et de composer une page de présentation simple, claire, normalisé et
évolutive.

Cette page vise notamment à montrer la distinction entre

  • les éléments de bloc (div, h1, ul...)
  • les éléments inline (img, a, span...)
Vérifier que vous avez bien indiqué l'encodage UTF-8 grace a la balise <meta>. (Attention, pour que la phrase précédente s'afiche correctement, il est
nécessaire d'utiliser des entiter HTML : cherchez sur le web de quoi il s'agit !)
